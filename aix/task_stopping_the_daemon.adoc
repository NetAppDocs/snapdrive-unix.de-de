= Beenden des Daemon
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Sie die Konfigurationsdatei SnapDrive für UNIX ändern, müssen Sie den Daemon anhalten und neu starten. Sie können den Dämon gewaltsam oder gewaltsam stoppen.
